# Caesar Cipher Tool

A standalone browser tool for encoding, decoding, and inspecting Caesar cipher text.

## Use

Open `index.html` in a browser. No install step or server is required.

## Features

- Encode and decode with shifts from 0 to 25
- Preserve case or output grouped five-letter blocks
- Copy, swap, clear, and load sample text
- View input letter frequency
- Preview all 26 brute-force decode candidates
