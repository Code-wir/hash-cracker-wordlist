# Contributing

Thanks for helping improve Wordlist Hash Cracker.

## Good First Contributions

- Add support for more hash formats
- Improve wordlist parsing and candidate transforms
- Add browser compatibility fixes
- Improve documentation and examples
- Add tests for hashing behavior

## Development

This project is plain HTML, CSS, and JavaScript. No install step is required.

Run the local checks with:

```sh
node --check hash-cracker-wordlist/app.js
node test/hash.test.mjs
```

## Responsible Use

Contributions should support defensive, educational, or authorized testing use cases. Do not add features intended to hide misuse, bypass rate limits, or target systems without permission.
